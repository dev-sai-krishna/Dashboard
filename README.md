# React Dashboard App

A modern and responsive dashboard built using React and Vite.
This project includes analytics cards, charts, customer reviews, and a clean user interface.

---

## Features

* Responsive sidebar with animation
* Dashboard cards for analytics overview
* Charts and statistics visualization
* Customer reviews section
* Data table display
* Fast performance using Vite

---

## Tech Stack

* React (Vite)
* CSS
* Framer Motion (animations)
* React Icons

## Installation and Setup

Clone the repository:

```
git clone https://github.com/dev-sai-krishna/Dashboard.git
```

Navigate to the project folder:

```
cd Dashboard
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
